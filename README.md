## Setup locally

1. Install ***Ruby***: [https://guides.rubygems.org/command-reference/#gem-install](https://guides.rubygems.org/command-reference/#gem-install),
2. Install ***Bundler***:
   `gem install bundler`,
3. Install dependencies (listed in the `Gemfile`):
   `bundle install`,
4. Build and serve the website locally:
   `bundle exec jekyll serve --config _config.yml,_config_dev.yml`.
